## Yellow reflection

Write 4-6 sentences:

```markdown
Prompt decision that mattered:
Context decision that mattered:
Harness decision that mattered:
One judgement I would repeat:
One thing I would do differently:
```

This is the field that makes Yellow Belt more than "another PR." It should show what you learned about using AI daily.

---

## Reviewer attestation

```markdown
Reviewer handle:
Reviewer role:
Review date:

I confirm that the evidence above satisfies Yellow Belt:
- White Belt evidence exists;
- Quest Y-1 produced a working utility and Three Pillars retro;
- Quest Y-2 produced a real practice log;
- Boss Fight Y-B has a merged PR or active named review per cohort rule;
- reflection names prompt, context, and harness decisions;
- no evidence includes sensitive material.
```

## Certification checklist

- [ ] White Belt tracker row exists.
- [ ] Quest Y-1 artefact runs and has a demo.
- [ ] Quest Y-1 retro names prompt, context, and harness.
- [ ] Quest Y-2 log exists with thirty honest entries.
- [ ] Boss Fight Y-B PR evidence exists.
- [ ] Yellow reflection exists.
- [ ] Reviewer attestation exists.
- [ ] Recertification due date is set.
- [ ] No sensitive details are present in the badge packet.

When all nine are checked, the Yellow Belt claim is ready for the tracker.
